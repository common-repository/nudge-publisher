Easily update and manage multiple Nudge tags across posts and pages.

== Description ==

This plugin allows publishers to manage the tracking on their native advertising content. Find out more at http://giveitanudge.com

Features:

* Upload multiple Nudge tags
* Add / Remove Nudge to Posts and Pages


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/nudge-publisher` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Nudge Publisher screen to start adding Nudge tags.
4. Go to a post or page you with to enable Nudge on, select the desired code and from the dropdown and hit update.
5. Visit http://check.ndg.io to make sure the code is firing correctly.
